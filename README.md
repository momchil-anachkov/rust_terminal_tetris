# Terminal Tetris

Tetris in the terminal. Written in Rust.

## Game Feature List:

- ✅ Basic tetris
- ✅ Wall kicks
- ✅ Ghost piece
- ✅ Smart rotations
- ✅ Next pieces sequence
- ✅ Hold piece
- ✅ Tick reset on slam
- ✅ Play/Pause
- ✅ Menu
- ❌ Line clear animation
- ❌ 1 second grace time before stick
- ❌ Incrementing speed/levels
- ❌ GIFs for each of the game features
- ❌ Score System
